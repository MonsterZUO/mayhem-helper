# MayhemHelper 用户手册（海克斯大乱斗助手）

给**国服海克斯大乱斗（ARAM: Mayhem）**玩家的桌面小工具：游戏内浮层看该英雄的海克斯优先级、选英雄时一键把出装导入客户端、离线速查任意英雄的海克斯/出装/套路。

> 只走官方 League 客户端接口（LCU）+ 系统置顶窗，**不注入内存、不读屏、不改游戏文件、不替你选三选一**。

---

## 一、下载与安装（Windows）

到 [Releases 页](https://github.com/MonsterZUO/mayhem-helper/releases/latest) 下载其一：

| 文件 | 说明 |
|---|---|
| `MayhemHelper_x.x.x_x64-setup.exe` | **推荐**，NSIS 安装包，双击一路下一步 |
| `MayhemHelper_x.x.x_x64_zh-CN.msi` | MSI 安装包，同样双击安装 |

- 仅支持 **Windows 10/11 64 位**（LoL 国服平台）。
- 首次运行如被 SmartScreen 拦（未签名），点「更多信息 → 仍要运行」。

---

## 二、准备工作（重要：游戏设为无边框）

游戏内浮层是**系统置顶窗**，盖不住「独占全屏」。用浮层前，把游戏设成**无边框**：

> 游戏内 `ESC → 选项 → 视频 → 窗口模式 → 无边框`

无边框和全屏在现代机上性能几乎无差别，大乱斗尤其无感。

## 三、三个功能怎么用

打开 MayhemHelper，登录着 LoL 客户端时它会自动连上。左侧栏第一项 **🎲 海克斯大乱斗** 是主界面。

### 功能 1：游戏内海克斯优先级浮层（核心）

1. 在「海克斯大乱斗」页点 **「切换游戏内浮层」** 打开浮层窗（半透明置顶小窗）。
2. 进入海克斯大乱斗、拿到英雄后，浮层**自动识别你这局的英雄**，显示：
   - **海克斯优先级**：按真实胜率排序，分棱彩 / 黄金 / 白银三档；
   - **最优三连套路**：整局三次选择的高胜率组合；
   - **核心出装**。
3. 游戏里等级 7/11/15 三选一时，拿浮层的排序**对照**你面前出现的 3 个选项，抢排得高的那个。

> ⚠️ 工具**不会**自动圈出「该选第几个」——官方接口不提供「本局给了你哪 3 个」的数据，这在技术上做不到（谁做都做不到，除非读屏/注入，本工具坚决不做）。它给的是「这英雄该优先抢哪些」的排序，你自己对照。

### 功能 2：选英雄时一键导入出装

在「海克斯大乱斗」页查到英雄后，点 **「一键导入出装到客户端」**：工具把该英雄的核心出装写成客户端**预设出装**，进游戏后在**商店**里就能看到、直接照着买。

- 只新增/更新本工具自己的一份出装，**不会动你已有的自定义出装**。
- 换英雄再导入会覆盖上一次本工具写的那份（不会越堆越多）。

### 功能 3：海克斯 / 出装速查

不打游戏也能查：在「海克斯大乱斗」页输入**英雄 ID**（如德邦=5）查询，看任意英雄的海克斯优先级、核心出装、三连套路。

> 当前版本用英雄 ID 查询（英雄名下拉选择在后续版本加）。常见英雄 ID 可网上搜「LOL 英雄 ID 对照表」。

## 四、数据从哪来（要点，务必看）

- 数据源是**外服（韩服 KR）**的真实战绩（海克斯/出装/套路胜率）。界面角标会标「**数据来源 KR · 版本 X**」。
- **为什么不是国服数据**：Riot 官方接口不开放海克斯大乱斗的增强数据，腾讯国服也没有开放统计——市面上所有同类数据都是外服。海克斯的**相对优先级**由游戏机制决定、跨服基本通用，所以外服数据作参考是可靠的；只是国服版本偶尔滞后时，绝对胜率数字会有点出入。
- **连不上数据怎么办**：工具内置了**打包好的 173 英雄出厂快照**。万一你的网络连不上外服数据源，会自动回退用快照（角标显示「KR·出厂」），仍然能用，只是不是最新实时数据。

## 五、常见问题（FAQ）

**Q：浮层打开了但游戏里看不到？**
A：把游戏视频设置改成**无边框**（见第二节）。独占全屏盖不住任何置顶窗，这是系统限制。

**Q：浮层不自动显示英雄 / 一直是空的？**
A：确认①LoL 客户端登录着、②工具显示已连接、③你已经在海克斯大乱斗里拿到英雄。选英雄阶段随机/重 roll/换英雄后浮层会跟着更新。

**Q：点了导入出装，游戏商店里没看到？**
A：确认是在**海克斯大乱斗**对局里看商店（出装按模式过滤）；重开一次商店界面刷新看看。若仍没有，把现象反馈到 [Issues](https://github.com/MonsterZUO/mayhem-helper/issues)。

**Q：数据一直显示「KR·出厂」/ 取不到实时数据？**
A：你的网络连不上外服数据源（国服家庭网络较常见）。工具已用出厂快照兜底，能正常用；想要实时数据需要能访问外服的网络环境。

**Q：有没有快捷键开关浮层？**
A：当前版本用界面按钮切换。全局快捷键在后续版本接入。

## 六、已知限制（诚实说明）

- **仅 Windows**：LoL 国服在 Windows，工具也只在 Windows 跑。
- **海克斯大乱斗是轮换模式**：模式没开放时，实时数据仍可查（速查库），但局内自动识别要等模式开放。
- **全局热键**未接入（用界面按钮切浮层）。
- **速查用英雄 ID**输入（英雄名选择待做）。
- 浮层的点击穿透、覆盖表现等在不同 Windows 环境可能需要微调——遇到问题欢迎反馈。

## 七、隐私与安全

- 只调用官方 League 客户端接口（LCU）读取对局信息、写入预设出装/符文，与 OP.GG、Blitz 桌面版同类原理。
- **不注入游戏内存、不 hook DirectX、不读屏、不改游戏文件。**
- 浮层是独立的系统置顶窗口，不接触游戏进程。
